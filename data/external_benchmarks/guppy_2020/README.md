# Fitzpatrick 2020 guppy genetic-rescue stress test

The 2009 translocations and waterfall barriers independently establish
downstream source-to-recipient flow in two drainages. The runner compares each
pre-flow recipient (NCA or NTY), its post-flow serial sample (PCA or PTY), and
the shared SGS source proxy. Candidate class C is biologically anchored, but
the proxy/serial-sample topology is not the simulation's exclusive one-edge
history and cannot yield a formal accuracy rate.

The source repository has no explicit license. VCFs and the audit script are
therefore fetched by pinned commit/hash at runtime and are never vendored. The
standard Caigual/Taylor panels contain 6,877/6,696 loci; strict
within-population-polymorphic sensitivities contain only 30/22 loci. Earlier
6,766/6,811 and 21/29 counts were not reproducible and are intentionally
rejected. The author release was filtered across pre/post populations, and both
runner filters inspect post-flow P2. The manipulation establishes direction
independently of the SNPs, but locus inclusion is not prospective held-out.
