# Mandeville 2019 Yellowstone cutthroat--rainbow trout stress test

The primary panel is frozen from non-genomic exposure metadata: SFOwlCreek
Yellowstone cutthroat trout P1, Trout Creek P2 (the sampled site nearest the
stocked Buffalo Bill Reservoir), and Story Hatchery rainbow trout P3. Candidate
class C is only an ecological introduction-history prior. Both trout taxa were
stocked, the sampled populations use fully disjoint sequencing-library sets,
and the modern references are proxies, so no panel supplies formal direction or
gate truth.

The runner audits source GT and a unique-PL-argmin representation, their locus
overlap, a Tensleep reference swap, a directly stocked Big Creek sensitivity,
and a same-species candidate-null structure diagnostic. It also records the
duplicated Dryad v2 archive listing, absent per-individual Entropy labels, and
the released converter's loss of informative PL values when GT is masked.
The source SNPs were globally call-rate/MAF ascertained using all fish, and the
later pooled/within-population filters also use panel labels. The corrigendum
DOI is recorded, but its publisher-hosted correction text was unavailable and
its substantive delta remains explicitly unaudited.
