# Weeks 2025 dingo pedigree panel

The runner deterministically materializes its 295-sample manifest from the
hash-pinned, headerless Figshare metadata: `alpine -> P1`, `back -> P2`, and
`dog -> P3`. The resulting LF-delimited manifest is 3,245 bytes with SHA-256
`12eed285869fa3426b82442ebbb44e866af77cf6013c7853776d421d20f23d45`.

No free-ranging individual inferred to be admixed is used as truth. The paper's
eight known captive `(dingo x dog) F1 x dingo` backcrosses independently anchor
a dog-introgressing component, whose candidate orientation is class C
(`P3 -> P2`). The cross is not an exclusive population-level single-edge
history, and the pooled references are proxies, so it is a guarded concordance
stress test rather than a formal accuracy datum. Desert and Mallee are not used
as controls because their counts/labels disagree across the paper and the two
released metadata files.
